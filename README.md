# 学術ポートフォリオサイト

Quartoで構築された学術ポートフォリオサイトです。大学院生の研究業績や経歴を紹介するためのウェブサイトとして設計されています。

## 概要

このリポジトリは、GitHub Pagesでホストされる個人の学術ポートフォリオサイトのソースコードを含んでいます。Quartoフレームワークを使用して構築されており、シンプルな更新と管理が可能です。

## 特徴

- 清潔でプロフェッショナルなデザイン
- 研究業績、出版物、経歴などの紹介ページ
- モバイルレスポンシブなレイアウト
- GitHub Pagesによる簡単なホスティング

## ローカルでの開発

### 前提条件

- [Quarto](https://quarto.org/docs/get-started/) がインストールされていること
- 基本的なマークダウンの知識

### サイトのローカルプレビュー

```bash
quarto preview
```

### サイトのビルド

```bash
quarto render
```

## デプロイ

このサイトはGitHub Pagesを使用してデプロイされます。メインブランチに変更をプッシュすると、GitHub Actionsによって自動的にサイトが構築され、デプロイされます。

## カスタマイズ方法

1. `_quarto.yml`でサイト全体の設定を変更
2. 各`.qmd`ファイルで各ページの内容を編集
3. `custom.scss`でスタイルをカスタマイズ
4. `images/`ディレクトリに画像を追加
